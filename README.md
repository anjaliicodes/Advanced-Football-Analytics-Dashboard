Football-Analytics-Dashboard
A professional-grade, responsive single-page telemetry dashboard built with HTML, CSS, and vanilla JavaScript. Designed to visualize advanced football team metrics (such as Expected Goals and Possession Control), this application features interactive data visualizations powered by Chart.js, a modern CSS Grid architecture, and simulated asynchronous live data syncing.

Key Features:
Advanced Data Visualization: Interactive line and doughnut charts using Chart.js to track performance trends and tactical distributions.

Modern Telemetry UI: A sleek, dark-themed aesthetic utilizing CSS variables, Phosphor Icons, and hover-state glassmorphism.

State Management & Async UI: Built with an internal AppState object for scalable tab navigation and simulated asynchronous API fetching with loading states.
